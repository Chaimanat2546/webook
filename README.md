# Project Name

Admin-only web app for managing pool villa images.

## Current focus

The current development focus is the house image management feature.

## Tech stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- Cloudflare Workers

## Getting started

1. Copy `.env.example` to `.env.local`
2. Fill required environment variables
3. Run dev server

## Scripts

- `npm run dev`
- `npm run typecheck`
- `npm run lint`
- `npm test`
- `npm run build`
